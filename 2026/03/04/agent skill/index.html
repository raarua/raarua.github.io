agent skill
是一个说明文档，用来教大模型“怎么做事”。

MCP 和 agent skill 的区别

  • MCP（Model Context Protocol）

    • 更像是「给大模型接工具 / 接数据源」。
    • 通过 MCP，可以让模型去访问数据库、HTTP 接口、本地文件等，把外部数据“拿进来”用。
    • 重点是：提供能力 / 数据通道，类似「给模型接各种 API」。

  • agent skill

    • 是一份纯文档（Markdown），写的是「在某个任务上，你应该按什么步骤做」。
    • 内容包括：任务背景、注意事项、推荐步骤、示例回答等。
    • 重点是：教模型怎么用现有能力来完成任务，类似「给模型写操作手册 / SOP」。

可以简单记：

  • MCP = 插件 / 工具 / 数据源（解决“能做什么 / 从哪拿数据”）
  • agent skill = 使用说明 / 工作手册（解决“拿到数据后要怎么处理、按什么流程做”）
