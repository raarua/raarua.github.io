记录一些常用的算法模板与思路，方便复习与比赛时查阅。

基本框架（C++）

常用 STL

  • vector：动态数组
  • set / map：去重、键值对
  • queue / stack：BFS、DFS
  • sort()：排序，可自定义比较函数

复杂度

  • 一般 1e8 次操作约 1 秒
  • 注意数据范围，选择合适的算法与数据结构
